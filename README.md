# рҹ“– About MobileTech

**MobileTech** is a **full-stack Django-based e-commerce web application** built to deliver a seamless online shopping experience for mobile devices.

The platform enables users to browse products, search instantly, manage shopping carts, and securely authenticate. Administrators can efficiently upload, update, and manage products through a dedicated dashboard.

Designed with a modern responsive interface, MobileTech demonstrates practical full-stack web development using Django, Bootstrap, JavaScript, and SQLite, and is deployed on **Railway** for easy public access.


```bash

```

---

## 2пёҸвғЈ Create a Virtual Environment

### Windows

```bash
python -m venv venv
```

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

---

## 3пёҸвғЈ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4пёҸвғЈ Apply Database Migrations

```bash
python manage.py makemigrations
```

```bash
python manage.py migrate
```

---

## 5пёҸвғЈ Create Admin Account

```bash
python manage.py createsuperuser
```

---

## 6пёҸвғЈ Start Development Server

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
